# html介绍

* hypertext market language
  * 超文本标记语言
* 主要形式
  * .htm
  * .html
* 文档结构组成

  * ```html
    <!DOCTYPE html>
    <html>
    <head>
        <meta charset="utf-8">
        <title>document</title>
    </head>
    <body>

    </body>
    </html>
    ```

* 属性

  * 属：给予元素基本标识

  * 性：样式or描述元素特征

  * 通用属性：id/class/style\(contenditable\)

  * 特有属性：colspan/type/value...
